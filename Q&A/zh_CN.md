# 欢迎来到Q&A
  大多数内容都会列在这里。

## 废话介绍
包含基础优化及 OptiFine 部分功能，让你玩的比较舒服的整合包。  
版本号格式为x.y.z:  
- 如果更新 Minecraft 版本，版本号将会重置。  
- **x** : 重要更改(大规模调整，功能性模组的增减)，  
- **y** : 小调整(优化模组的增减)。
- **z** : bug修复或模组更新。

## 环境要求
- **Java 22 或以上版本**  
  > 如果不想安装 Java 22 或以上版本，请删除模组 **C2ME**，这样仍然需要 Java 21 或以上版本。  
- **推荐安装 Java 25**：自 **26.1-snapshot-1** 之后的游戏版本都需要 Java 25。  
- 如果你是手机用户，出现问题请自行排查。

## 安装方法
1. (推荐)直接在启动器中搜索安装。
2. 下载整合包后 (**mrpack文件**) ，将其直接拖入启动器窗口。  
3. 如果无法拖入安装，可手动选择：  
  - **PCL2**: 下载 → 整合包 → 安装已有整合包  
  - **HMCL**: 下载 → 整合包 → 安装整合包 → 导入本地整合包  
  - 其它启动器大致相同。

## 模组提示
> 如果遇到不懂、不会用的模组，可以去 [MC百科](https://www.mcmod.cn) 查找资料。  

1. **C2ME (地形生成优化)**  
   需要 Java 22 或以上版本。  

2. **ViaFabricPlus (版本转译)**  
   游戏版本过新，为保证能正常游玩部分旧服务器，安装了 ViaFabricPlus。  
   但在低性能设备上非常不友好（版本转译消耗 CPU 资源）。  
   如果游玩需要转译版本的服务器过度卡顿，请选择对应的 Minecraft 版本游玩。  

3. **DistantHorizons (遥远的地平线)**  
   优化模组，超远视距只是额外功能。  
   可将该模组视距设置为 32，原版视距拉到 12 或更低，以降低远处地形质量进行优化。  
   对核显比较友好；如果 CPU 性能过差，请删除，否则可能是负优化。  

4. **caramelChat (输入法模组)**  
   Windows 用户可跳过。  
   部分 Linux、macOS 不兼容，可能导致系统卡死，请删除。  
   可查看该模组的 [GitHub 页面](https://github.com/LemonCaramel/caramelChat) 进行系统设置。  

### QA
- **Q: 为什么不用 Voxy 而是 DistantHorizons?**  
  **A:** Voxy 目前不支持大部分 DistantHorizons 的功能，对新手较麻烦。  

- **Q: 为什么不添加 BadOptimizations?**  
  **A:** 优化导致天空卡顿，破坏游戏沉浸感。  

- **Q: 为什么不添加 Krypton?**  
  **A:** Krypton 是服务端网络堆栈优化模组。测试发现服务端和客户端一起安装会导致随机丢包。  
  简单的朋友联机人数较少，优化效果不明显。  

- **Q: 为什么不添加 FerriteCore?**  
  **A:** 内存优化较激进，适合内存严重不足的情况。除非分配内存低于 2G，否则优化效果不明显且帧数降低。  

## 材质包提示
包含两个材质包，都是对模组进行汉化的。  

- **Cull Leaves (树叶渲染优化)**  
  提供了一个材质包（名为 `cullleaves/smartleaves`），整合包默认安装，  
  让树叶看起来不那么丑。  

## 按键提示
- **C (按住)** – 放大画面  
- **O** – 光影包选择界面  
- **K** – 启用/禁用光影  
- **R** – 重新加载光影  
- 其余按键皆为原版设置
